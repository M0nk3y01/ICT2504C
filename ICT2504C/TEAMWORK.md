# TEAMWORK CONTRIBUTION

This project was developed collaboratively by two team members.
Responsibilities were divided based on system modules and implementation
areas.

------------------------------------------------------------------------

## Member 1 -- Backend & Security Implementation

Responsibilities:

-   Designed backend architecture using Node.js and Express
-   Implemented authentication system using JWT
-   Implemented refresh token rotation
-   Implemented brute‑force login protection and account lockout
-   Developed password reset functionality
-   Implemented audit logging for security events
-   Designed and implemented database models using Sequelize ORM
-   Implemented role‑based access control
-   Developed REST API endpoints for authentication and user management
-   Implemented admin account unlocking functionality

------------------------------------------------------------------------

## Member 2 -- Frontend & User Interface

Responsibilities:

-   Developed React frontend using Vite
-   Implemented navigation system and routing
-   Built login and dashboard interface
-   Developed employee leave application interface
-   Implemented leave history display
-   Built manager leave approval interface
-   Implemented admin pages for employee creation and user management
-   Implemented role‑based UI rendering
-   Designed UI layout and styling
-   Integrated frontend with backend APIs

------------------------------------------------------------------------
